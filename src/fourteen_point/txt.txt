1. Установка необходимых зависимостей:
   - Установлены Node.js и npm.
   - Создан файл package.json и установлены зависимости: express, sequelize, sqlite3.

2. Создание структуры проекта:
   - Созданы директории: config, models, controllers, routes.
   - Созданы файлы для конфигурации базы данных, моделей, контроллеров, маршрутов.

3. Настройка базы данных:
   - В файле config/database.js настроено подключение к базе данных SQLite с использованием Sequelize.

4. Определение моделей:
   - Созданы модели Company, Product, Coach, Team, Student, Course, Enrolment в директории models.
   - В файле models/index.js установлены связи между моделями:
     - Один-ко-многим: Company и Product.
     - Один-к-одному: Coach и Team.
     - Многие-ко-многим: Student и Course через Enrolment.

5. Реализация основных операций:
   - В файле controllers/exampleController.js реализована функция для получения всех продуктов компании.
   - В файле routes/exampleRoutes.js настроен маршрут для вызова контроллера.

6. Запуск сервера:
   - В файле server.js настроен Express-сервер и синхронизация базы данных.
   - Запуск сервера осуществляется командой `npm start`, сервер слушает на порту 3000.

7. Проверка работы:
   - Проверено создание и получение данных через маршруты API.
   - Убедился в корректной работе связей и операций с базой данных.
