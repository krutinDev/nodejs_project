Отчет по выполнению задания:

- Файл app.js создан и содержит основной код для запуска сервера.
- В app.js подключен модуль http с помощью const http = require("http");.
- Сервер создан с использованием http.createServer() и обрабатывает входящие запросы.
- Сервер запускается с помощью команды node app.js.
- После запуска сервера можно открыть браузер и перейти по адресу http://localhost:3000/.
- Код выполняется командой node app.js в командной строке.
- Имя пользователя получено с помощью os.userInfo().username и используется в приветствии.
- Модуль sinModule.js содержит стрелочную функцию для вычисления синуса числа x.
- Дата и время запроса выводятся в командной строке с помощью console.log.
- Приветствие с именем пользователя ПК выводится с учетом времени суток.
- В файле names.js определены имена, которые используются в приветствии.
- Модуль greeting.js использует кэширование, чтобы приветствовать двух людей из names.js.
- Глобальная переменная date установлена с помощью global.date = new Date();.
- Глобальная переменная name получена из файла конфигурации и установлена с помощью global.name = config.name;.
- Глобальная переменная name выводится на консоль с помощью console.log.
