Отчет по выполнению задания:

1. Создание проекта:
    - Создана структура проекта с основными файлами: package.json, .env, db.js, index.js.

2. Установка зависимостей:
    - Установлены необходимые пакеты:
        - mysql2 для работы с MySQL.
        - dotenv для управления переменными окружения.

3. Настройка конфигурации:
    Создан файл .env для конфигурации подключения к базе данных с переменными:
        - DB_HOST: адрес сервера базы данных.
        - DB_USER: имя пользователя.
        - DB_PASSWORD: пароль пользователя.
        - DB_NAME: имя базы данных.
        - DB_CONNECTION_LIMIT: лимит подключений в пуле.

4. Создание пула подключений:
    - В файле db.js настроен пул подключений с использованием mysql.createPool() и параметрами из .env.

5. Выполнение запросов:
    - В index.js реализован асинхронный запрос к таблице users с использованием промисов.
    - Вывод результатов запроса в консоль.

6. Корректное завершение работы:
    - Реализовано закрытие пула подключений с помощью pool.end() в блоке finally.

7. Запуск и проверка:
    - Проект запущен командой npm start.
    - Успешно выполнен запрос к базе данных, результаты выведены в консоль.
